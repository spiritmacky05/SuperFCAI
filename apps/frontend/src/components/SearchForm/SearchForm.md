# SearchForm

The `SearchForm` component captures user inputs for fire safety analysis.

## Props

- `params`: Current search parameters.
- `setParams`: Dispatcher for search parameters.
- `onSubmit`: Submit handler.
- `isLoading`: Loading state indicator.

## Usage

```tsx
<SearchForm
  params={params}
  setParams={setParams}
  onSubmit={handleSubmit}
  isLoading={false}
/>
```
