# Logo

The `Logo` component renders the Super FC AI official emblem with a pulse animation and outer glow effect.

## Props

- `className`: Optional. Tailwind classes for the outer container.
- `size`: Optional. Options: `'sm' | 'md' | 'lg' | 'xl' | '2xl'`. Default is `'md'`.

## Usage

```tsx
<Logo size="lg" className="animate-bounce" />
```
