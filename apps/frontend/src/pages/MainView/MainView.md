# MainView

The `MainView` component represents the core dashboard layout of the application where users can input establishment parameters, generate reports using the AI, and interact with the assistant or ChatBox.

## Props

- `user`: Required. The currently logged-in User profile.
- `reportGen`: Required. An object exported from the `useReportGenerator` hook containing the parameters, result state, loading state, and the submission function.
- `setIsAssistantOpen`: Required. A boolean toggler to trigger the AI Assistant Modal.

## Dependencies

- `SearchForm`: Renders the inputs for the query.
- `ResultDisplay`: Renders the markdown result generated by the AI.
- `ChatBox`: Handles the interactive Q&A regarding the report.
- `NTCGenerator`: UI to handle the Notice to Comply.
- `AdComponent`: Interstitial Ads specifically shown only to Free users.

## Usage

```tsx
<MainView
  user={user}
  reportGen={reportGen}
  setIsAssistantOpen={setIsAssistantOpen}
/>
```
